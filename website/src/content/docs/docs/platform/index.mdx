The Distr Distribution Platform provides the tools to manage the commercial and operational aspects of your software distribution, from licensing and entitlements to customer management and compliance.

## License Management

Control access to your applications and artifacts:

- **[License Management](/docs/platform/license-management/)** - Overview of the licensing system
- **[Application Entitlements](/docs/platform/application-entitlements/)** - Control which application versions customers can deploy
- **[Artifact Entitlements](/docs/platform/artifact-entitlements/)** - Control which artifacts customers can access
- **[License Keys](/docs/platform/license-keys/)** - Issue signed tokens for usage limits and feature access

## Support

Validate, scan, and support your deployments:

- **[Support Bundles](/docs/platform/support-bundles/)** - Collect and review diagnostic data from customers
- **[Kubernetes Compatibility Matrix](/docs/platform/kubernetes-compatibility-matrix/)** - Validate Helm charts against multiple Kubernetes versions
- **[Vulnerability Scanning](/docs/platform/vulnerability-scanning/)** - Scan for known vulnerabilities in dependencies and images

## Customer Portal

The customer-facing interface for your end customers:

- **[Customer Portal](/docs/platform/customer-portal/)** - Overview of the white-labeled customer UI
- **[Artifact Registry](/docs/platform/customer-portal/registry/)** - End-customer guide to accessing artifacts
- **[Deployments](/docs/platform/customer-portal/deployments/)** - End-customer guide to deploying with agents
- **[Licenses](/docs/platform/customer-portal/licenses/)** - End-customer guide to viewing license keys
- **[Support Bundles](/docs/platform/customer-portal/support/)** - End-customer guide to creating support bundles
- **[Secrets](/docs/platform/customer-portal/secrets/)** - End-customer guide to managing secrets

## Customer Management

Manage customer organizations and their access:

- **[Customer Management](/docs/platform/customer-management/)** - Create customers, invite users, manage secrets and features
- **[Branding & White-Labeling](/docs/platform/branding/)** - Customize the customer portal with your branding

## User Management

Control user roles and permissions:

- **[Role-Based Access Control (RBAC)](/docs/platform/rbac/)** - Manage roles and permissions across organizations
